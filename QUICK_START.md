## في 5 دقائق فقط

### 1. تثبيت Flutter (إن لم يكن مُثبَّتاً)

```bash
# Linux/macOS
git clone https://github.com/flutter/flutter.git -b stable ~/flutter
echo 'export PATH="$PATH:$HOME/flutter/bin"' >> ~/.bashrc
source ~/.bashrc
flutter doctor
```

---

### 2. فتح المشروع وتشغيله

```bash
# داخل مجلد mavix_flutter
cd mavix_flutter

# تحميل المكتبات
flutter pub get

# تشغيل مباشرة (يفتح على المحاكي أو الجهاز المتصل)
flutter run
```

> **💡 نصيحة:** إذا لم تكن الخطوط محمّلة، سيستخدم التطبيق الخط الاحتياطي تلقائياً.

### 4. إعداد الخادم السحابي (اختياري للتجربة المحلية)

للعمل **بدون** خادم (محلي فقط):
- الدخول كضيف ✅
- تسجيل تمارين ✅
- تتبع قياسات ✅
- كل البيانات تُحفظ على الجهاز ✅

للعمل **مع** المزامنة السحابية:
1. أنشئ مشروع Google Apps Script
2. ارفع كود الـ backend
3. انسخ رابط النشر
4. أدخله في: **الإعدادات → رابط الخادم**

### 5. تدفق الاستخدام اليومي

```
📱 فتح التطبيق
    ↓
🏠 الرئيسية — ملخص اليوم
    ↓
🏋️ التمرين — اختر جلسة → ابدأ → سجّل الأوزان
    ↓
🥗 التغذية — أضف وجباتك للتتبع
    ↓
📅 الحضور — سجّل حضورك (الأدمن فقط)
    ↓
📊 التقدم — راجع 1RM والقياسات
    ↓
💬 الدردشة — تواصل مع المدرب
```

### 6. شاشات التطبيق الرئيسية

#### 🏠 الرئيسية
- ملخص اليوم (سعرات + تمارين)
- إحصائيات سريعة (الأسبوع/البروتين)
- آخر التمارين المسجلة
- روابط للدردشة والإعدادات

#### 🏋️ التمرين
- عرض برنامجك الحالي
- جلسات الأسبوع مع زر "ابدأ"
- **داخل الجلسة:**
  - إحماء موصى به (Pallof Press، إلخ)
  - كل تمرين مع حقول الوزن والتكرار
  - زر ✓ لتأكيد كل ست
  - مؤقت للجلسة
  - حفظ + مزامنة تلقائية

#### 🥗 التغذية
- نسبة استهلاك السعرات اليومية
- ماكرو البروتين/الكارب/الدهون
- **حاسبة سعرات ذكية** (BMR + TDEE)
- إضافة أطعمة من قاعدة بيانات 20+ طعام عربي

#### 📅 الحضور
- تقويم شهري تفاعلي
- رموز: ✓حضور ✗غياب ~تأخير E عذر ★إجازة
- إحصائيات شهرية + نسبة الحضور
- الأدمن يضع العلامات، المستخدم يشاهد

#### 📊 التقدم
- **قياسات جسدية:** وزن، دهون%, عضلات, مقاسات
- **تقدم التمارين:** أفضل 1RM لكل تمرين مع مقارنة
- يكشف الركود تلقائياً (3 أسابيع بدون تقدم)

#### ⚙️ الإعدادات
- تبديل الوضع الداكن/الفاتح
- معلومات الاشتراك وتاريخ الانتهاء
- إعداد رابط GAS
- تغيير البرنامج والبيانات الجسدية
- تسجيل الخروج

### 7. بناء APK للتثبيت

```bash
# من داخل mavix_flutter/
flutter build apk --release --split-per-abi

# ستجد الـ APK في:
# build/app/outputs/flutter-apk/app-arm64-v8a-release.apk
# (هذا للأجهزة الحديثة — الأكثر شيوعاً)
```

**حجم الـ APK المتوقع:** 8-12 MB

### 8. استكشاف الأخطاء السريع

| المشكلة | الحل السريع |
|---------|-------------|
| تطبيق لا يفتح | `flutter clean && flutter pub get && flutter run` |
| تجميد عند Splash | تحقق من `flutter doctor` |
| لا يتصل بالخادم | تحقق من رابط GAS في الإعدادات |
| بيانات لا تُحفظ | تحقق من صلاحيات التخزين |
| خطأ في Gradle | `cd android && ./gradlew --stop && flutter build apk` |

### 9. هيكل البيانات المحلية

```
SQLite (mazen_coach.db)
├── users          — بيانات المستخدمين
├── workout_logs   — سجلات التمارين
├── meals          — وجبات التغذية
├── attendance     — سجل الحضور الشهري
├── measurements   — القياسات الجسدية
├── chat_messages  — رسائل الدردشة
├── notifications  — الإشعارات
├── sync_queue     — طابور المزامنة
└── kv             — بيانات عامة (مفتاح-قيمة)
```

## التطوير الإضافي المقترح

1. **دفع الإشعارات** — Firebase Cloud Messaging
2. **تحميل الصور** — تحميل صورة إيصال الدفع
3. **Admin Panel كامل** — شاشة إدارة الأعضاء والاشتراكات
4. **رسوم بيانية** — fl_chart لرسم تقدم الوزن والـ 1RM
5. **نسخ احتياطي** — تصدير البيانات كـ JSON/Excel
6. **برنامج مخصص** — واجهة إنشاء تمارين وجلسات من الصفر
7. **iOS Build** — إضافة ملفات Xcode و Info.plist
